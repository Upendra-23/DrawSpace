# DrawSpace

A feature-rich drawing application built with React, TypeScript, Zustand, and HTML5 Canvas.

## Features

- **Drawing Tools**: Pen, shapes (rectangle, circle, line, arrow, triangle), laser pointer, eraser, select, and move tools
- **Canvas Controls**: Zoom in/out, pan with spacebar+drag, responsive canvas
- **Selection & Editing**: Drag-select elements, cut/copy/paste with keyboard shortcuts or toolbar buttons
- **Undo/Redo**: Full history management with Ctrl+Z / Ctrl+Y
- **Customization**: Color picker with presets, adjustable stroke width, shape selector
- **Dark/Light Theme**: Toggle between dark and light mode
- **Glassmorphism UI**: Modern translucent toolbar design
- **Grid Overlay**: Dot grid helper for alignment

## Tech Stack

- **React 19** with TypeScript
- **Zustand** for state management
- **HTML5 Canvas** for rendering (not SVG/DOM)
- **Vite** for fast development and bundling
- **ESLint** for code quality

## Getting Started

```bash
npm install
npm run dev
```

## Build

npm run build
